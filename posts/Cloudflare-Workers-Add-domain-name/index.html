Cloudflare Workers添加自定义域名和修改Heroku节点

发表于2022-05-09|更新于2022-05-11|在线鼓捣Web
|字数总计:185|阅读时长:1分钟|阅读量:

介于5月8日晚CloudFlare Workers 的业务域名 Workers.dev 被防火长城 DNS 污染。

故而教一下如何在Cloudflare Workers添加自定义域名

欢迎加入Telegram群聊讨论https://t.me/iflugp

添加域名

在Cloudflare中添加自己的域名

（这一步不做赘述）

添加自定义域名

​ 打开Workers里你的应用，选择触发器选项，点击添加路由

​ 区域选择你在Cloudflare托管的域名，路由填写域名xxx（前缀）.xxxx（域名）/*

​ 最后添加路由，即可通过xxx（前缀）.xxxx（域名）访问你的项目

配置VMess节点

地址=cloudflare优选IP或者域名

伪装域名(Host)=自定义域名(xxx.xxxx.xx)

SNI=自定义域名(xxx.xxxx.xx)
