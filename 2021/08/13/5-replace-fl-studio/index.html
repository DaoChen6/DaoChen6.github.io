5个在Linux中代替FL Studio的编曲软件

FL Studio是 Windows 和 Mac 平台的强大数字音频工作站和音乐创作工具。它是商业软件，被认为是当今最好的音乐制作程序之一。但是，FL Studio不支持在Linux上运行，将来也不打算提供支持。所以，如果你刚刚切换到Linux平台，想要编曲，你需要选择一个好的编曲工具。这里有5个可以在Linux上运行的编曲软件代替FL Studio！

1. LMMS

LMMS 是一个开源音乐创作工具，允许用户制作旋律、节拍和其他类型的音乐，很像 FL Studio。LMMS 工具支持 MIDI 测序、采样、音乐硬件支持，是 FL Studio 用户切换到 Linux 的完美免费替代品。

显著功能

• LMMS 的用户界面与 FL Studio 非常相似，支持许多相同的功能，例如简单的节拍制作器和大量虚拟文书。

• LMMS 对硬件（包括 MIDI 键盘）有出色的支持。

• LMMS 为 VST 仪器提供内置支持，包括 32 位 VST 桥。

• LMMS 包括 16 个内置合成器，包括Nintendo Game Boy, NES, Commodore 64 等等。

下载

LMMS 是开源的，因此，可用于在每个 Linux 分支上安装。要使其在 Linux 上工作，请打开终端，并在包管理器搜索”LMMS”。如果您需要帮助安装LMMS应用程序，[请点此查看 ](LMMS | Home)。

2. T7 Daw

T7 Daw 是一个免费增值数字音频工作站，适用于 Windows、Mac 和 Linux。该应用程序（购买时）提供了一些竞争激烈的音乐创作功能和工具，将熟悉的FL Studio老兵，如各种合成器，数字乐器，MIDI支持，和壮观的音频编辑套件。

显著功能

• T7 Daw 是跨平台的，用户将在 Linux、Mac 和 Windows 上获得相同的数字音频工作站/音乐制作体验。

• T7 Daw支持无限的音频和 MIDI 曲目。

• 可以与视频轨道同步音频。

• 配备自动化工具，使您的音乐创作过程更加简单。

下载

T7 Daw是付费软件（现在有免费版，自行前往官网查看）。因此，如果您有兴趣在 Linux 平台上使用它，则无法通过传统方法安装它。相反，前往官方网站购买和下载软件。

3. Bitwig Studio

Bitwig Studio是一种商业音乐工具，可用于音乐创作。Bitwig Studio配备了 150 多个音乐模块，并配备了设计师和艺术家喜爱的功能。此外，Bitwig Studio还提供 DJ 功能，这是有用的，特别是对于那些谁执行他们的音乐现场。

显著功能

• 一个功能齐全的数字音频工作站，支持Windows，Mac和Linux，确保你得到相同的体验，无论你运行它是什么操作系统。

• 在音乐创作过程中，有超过10 GB的声音需要处理。

• 船舶超过80种乐器和效果，以帮助您的创造力在音乐制作过程中。

• Bitwig Studio支持声音设计、录音和现场表演。

下载

Bitwig Studio不是免费软件，有大量用户购买它使用它。要获取您的 Bitwig Studio副本，请访问Bitwig Studio官方网站，然后单击”下载”按钮。选择”下载”按钮后，您将看到获取 Linux 应用的选项，以及有关如何注册许可证的说明。

4. Ardour

Ardour是Mac、Linux和Windows的数字音频工作站。有了它，用户可以录制音频，混合音频等。一般来说，Ardour 更注重混音和录制，而不是创建节拍或与数字仪器、合成器等（如 FL Studio）配合使用，但它具有强大的插件支持。

显著功能

• Ardour 对 Linux 上的 JACK 音频系统有着出色的支持，该系统非常坚固，允许用户使用录制的音频进行更高级的处理。

• Ardour 支持无限多通道曲目，确保您的编辑和制作可以像您喜欢的那样大或复杂。

• Ardour 可以直接导入视频和提取声音。

• Ardour 支持插件，以便用户可以添加类似于 FL Studio 的功能。

下载

像许多其他节目在这个列表中谈论，Ardour是开源和免费的。由于是开源的，Ardour 通过传统软件源广泛提供给所有 Linux 操作系统。要在系统上安装 Ardour，请打开终端窗口，搜索”Ardour”，并按您通常安装应用程序的方式安装软件。或者，前往Ardour官网，了解如何让 Ardour 安装到您的系统上。

5. Rosegarden

Rosegarden 是一个通用的音频/MIDI测序器，和乐谱编辑。它是一个免费的应用程序，只支持Linux平台，并有一个简单，易于学习的用户界面，一定会吸引初学者。如果你想使用 FL Studio 制作音乐，可以先试试 Rosegarden 。

显著功能

• Rosegarden 有一个易于使用的界面，欢迎新的用户。

• Rosegarden 支持编辑、创作和修改乐谱。

• Rosegarden 附带包含合成器，可用于播放MIDI轨道通过。

下载

Rosegarden 是开源的，只提供给Linux用户，所以安装它的包在所有流行的Linux操作系统携带。要安装它，启动一个终端，搜索”Rosegarden”，并以你通常安装应用程序的方式安装软件。或者，查看Rosegarden官网。

结论

FL Studio是一个精心设计的编曲工具，也是同类编曲工具中最好的编曲工具之一。可悲的是，Linux 用户将永远不会体验到它提供的功能。值得庆幸的是，Linux 上还有大量其他有用的编曲工具，这些工具提供了类似的功能。
